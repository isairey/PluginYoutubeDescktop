# 🤝 Contribuciones

Las contribuciones son bienvenidas ❤️

Este proyecto Open Source está abierto a mejoras, correcciones y nuevas funcionalidades desarrolladas por la comunidad.

---

# 🚀 Cómo contribuir

## 1️⃣ Haz Fork del repositorio

```bash
git clone https://github.com/usuario/proyecto.git
```

---

## 2️⃣ Crea una nueva rama

```bash
git checkout -b feature/nueva-funcion
```

---

## 3️⃣ Realiza tus cambios

Puedes contribuir con:

- ✨ Nuevas funcionalidades
- 🐛 Corrección de errores
- ⚡ Optimización de rendimiento
- 🎨 Mejoras visuales
- 📚 Documentación
- 🔧 Refactorización de código

---

## 4️⃣ Haz commit siguiendo Conventional Commits

```bash
git commit -m "feat: nueva funcionalidad"
```

Ejemplos válidos:

```bash
feat: add media controls
fix: resolve websocket issue
docs: update README
refactor: improve plugin architecture
```

---

## 5️⃣ Haz push a tu rama

```bash
git push origin feature/nueva-funcion
```

---

## 6️⃣ Abre un Pull Request 🚀

Describe claramente:

- Qué problema resuelve
- Qué cambios realizaste
- Cómo probar la funcionalidad
- Capturas si aplica

---
