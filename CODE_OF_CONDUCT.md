# 🌍 Nuestro Compromiso

Como miembros, colaboradores y líderes de esta comunidad, nos comprometemos a mantener un entorno:

- 🤝 Respetuoso
- 🌎 Inclusivo
- 🚀 Profesional
- ❤️ Amigable
- 🔥 Libre de acoso
- ⚡ Seguro para todos

Independientemente de:

- Edad
- Nacionalidad
- Género
- Experiencia
- Apariencia
- Religión
- Identidad
- Nivel educativo
- Situación económica
- Orientación sexual
- Discapacidad visible o invisible

Nuestro objetivo es fomentar una comunidad saludable y positiva donde todos puedan participar cómodamente.

---

# ✨ Estándares de la Comunidad

## ✅ Comportamientos Esperados

Se consideran acciones positivas:

- 🤝 Mostrar empatía y respeto
- 🎯 Aceptar críticas constructivas
- 📚 Compartir conocimiento
- 🚀 Ayudar a otros colaboradores
- 💬 Mantener conversaciones profesionales
- ❤️ Promover un ambiente amigable
- ⚡ Aprender de los errores

---

## ❌ Comportamientos Inaceptables

No se permitirá:

- 🚫 Acoso público o privado
- 🚫 Lenguaje ofensivo o discriminatorio
- 🚫 Ataques personales o políticos
- 🚫 Trolling o provocaciones
- 🚫 Publicar información privada sin permiso
- 🚫 Conductas agresivas o intimidatorias
- 🚫 Comentarios degradantes o irrespetuosos

---

# 🛡️ Aplicación y Moderación

## ⚖️ Responsabilidades de los Moderadores

Los líderes y moderadores del proyecto tienen la responsabilidad de:

- 🔍 Revisar reportes
- ⚡ Actuar de forma justa
- 🤝 Mantener el orden
- 🚀 Garantizar un ambiente sano
- 🛡️ Proteger a la comunidad

También podrán:

- Eliminar contenido inapropiado
- Editar comentarios ofensivos
- Rechazar contribuciones dañinas
- Suspender usuarios problemáticos

---

# 🌐 Alcance

Este Código de Conducta aplica en:

- GitHub
- Pull Requests
- Issues
- Discussions
- Chats comunitarios
- Redes sociales oficiales
- Eventos online y presenciales

También aplica a cualquier persona que represente oficialmente al proyecto.

---

# 📢 Reportar Problemas

## 📩 Contacto de Moderación

Si encuentras comportamientos abusivos, ofensivos o inapropiados, puedes reportarlos a:

```txt
Developer.SR.IRP@gmail.com
```

Todos los reportes serán:

- 🔒 Confidenciales
- ⚡ Revisados rápidamente
- 🤝 Investigados de forma justa
- 🛡️ Tratados con privacidad y seguridad

# ⚠️ Guía de Sanciones

## 🟡 1. Corrección

### 📌 Ejemplo

- Lenguaje inapropiado
- Conducta poco profesional

### ⚡ Consecuencia

- Advertencia privada
- Explicación de la infracción
- Solicitud de corrección

---

## 🟠 2. Advertencia Formal

### 📌 Ejemplo

- Conducta reiterada
- Comentarios ofensivos

### ⚡ Consecuencia

- Restricción temporal
- Prohibición de interacción
- Supervisión comunitaria

---

## 🔴 3. Suspensión Temporal

### 📌 Ejemplo

- Acoso reiterado
- Violaciones graves

### ⚡ Consecuencia

- Suspensión temporal
- Restricción total de participación
- Bloqueo de canales comunitarios

---

## ⛔ 4. Expulsión Permanente

### 📌 Ejemplo

- Acoso severo
- Amenazas
- Conductas agresivas constantes

### ⚡ Consecuencia

- Ban permanente
- Eliminación de acceso comunitario
- Restricción total de interacción
